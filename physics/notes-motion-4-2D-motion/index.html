2-D motion

Solving for 2-d motion uses the same techniques as in 1-D, but with added complexity so we need to track our variables better. We will organize our information in an x column and y column with time separate. The x information and y information are independent of each other except for time which they share. Lets try an example problem. Click to watch again.

Example: A ball moving horizontally at 2m/s rolls off a table that is 1.5m high. Find how far the ball travels horizontally before it hits the ground.

Here is the information we know from reading problem

Δt = ?
x: horizontal (left/right) y: vertical (up/down)
Δx = ? Δy = 1.5m
vi = 2m/s vi = 0
vf = ? vf = ?
a = 0 a = 9.8 m/s2

There isn't enough information to solve on the horizontal side. Let try the vertical.

$$v_{f}^{2} = v_{i}^{2}+2a \Delta y$$ $$v_{f}^{2} = 0^{2}+2(9.8)(1.5)$$ $$v_{f}^{2} = 24.9$$ $$v_{f} = 5.42 \small\frac{m}{s}$$
$$v_{f} = v_{i}+a \Delta t$$ $$5.42 = 0+(9.8)\Delta t$$ $$0.55s = \Delta t$$

With this new information we can use time to solve on the horizontal side.

Δt = 0.55s
x: horizontal (left/right) y: vertical (up/down)
Δx = ? Δy = 1.5m
vi = 2m/s vi = 0
vf = ? vf = 5.42m/s
a = 0 a = 9.8 m/s2
$$\Delta x = v_{i}\Delta t + \small\frac{1}{2}a \Delta t^{2}$$ $$\Delta x = 2(0.55) + \small\frac{1}{2}(0)(0.55)^{2}$$ $$\Delta x = 1.1m$$
